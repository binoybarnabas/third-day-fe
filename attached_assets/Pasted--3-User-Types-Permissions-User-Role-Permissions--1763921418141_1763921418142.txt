### 5. Detailed User Flows

#### A. New User Journey (First Purchase)
1. Lands on homepage → browses Men/Women → clicks product → PDP
2. Clicks “Add to Cart” → mini-cart pops up → “View Cart” or “Checkout”
3. Checkout → “Continue as Guest” or “Login/Signup”
4. Chooses “Sign up with Google” or “Phone OTP”
   - Phone OTP flow: Enter phone → OTP → Auto-fill name (optional) → proceed
5. Shipping address form (with pincode checker for delivery feasibility)
6. Payment page → Razorpay (UPI QR, UPI ID, Cards, Wallets)
7. Order success → Order confirmation email + SMS with tracking link
8. Account automatically created → can see order in “My Orders”

#### B. Returning User Journey
1. Clicks “Login” top-right → Google or Phone → lands on homepage
2. Profile icon shows name/initials
3. Can directly go to Wishlist, Cart, My Orders

#### C. Wishlist / Favourites
- Heart icon on PLP and PDP
- Clicking when logged in → adds to wishlist
- Clicking when not logged in → prompts login → then adds

#### D. Cart Behaviour
- Persists even if user closes browser (saved in database for logged-in, localStorage for guest)
- Guest cart merges with account cart after login
